# Lunetix Browser Engine

Modern web browser built on Chromium with memory optimization and enhanced privacy.

## Features

- **Memory Optimizer** - Automatic tab suspension to save RAM
- **Privacy-focused** - Independent from Chrome/Google
- **Chromium-based** - Full web compatibility

## Build

```cmd
setup_lunetix.bat
build_lunetix.bat
```

## Requirements

- Windows 10/11
- Python 3.7+
- Visual Studio 2022
- ~15GB disk space

Chromium fork with patch-based architecture for maintainable browser development.
